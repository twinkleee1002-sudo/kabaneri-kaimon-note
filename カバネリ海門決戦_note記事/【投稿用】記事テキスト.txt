① 天井短縮の仕組みと判別法
━━━━━━━━━━

カバネリ海門決戦の最大の特徴は「天井短縮」です。

通常時の天井はゲーム数天井996G（6周期目）ですが、以下の条件を満たすと天井が596G（4周期目）に短縮されます。

・ST駆け抜け後（STに突入したがATに繋がらず終了した場合）
・景之ST終了後（上位STが終了した場合）
・設定変更後（リセット後）

ここがガチで大事です。
天井短縮中は設定1でも出玉率が100%を超えます。
つまり0Gから打っても理論上プラスになる状態です。

一般人は「400Gハマってるから拾おう」と単純にG数だけで判断しますが、プロは「この台は596G短縮なのか996G通常なのか」を最初に判別します。

同じ400Gハマりでも——
596G短縮台なら残り約196Gで天井。期待値は+6,000円。
996G通常台なら残り596Gもあり、期待値はたった+200円。

この差を見逃すかどうかで月収が15万円以上変わります。

▶ 判別の具体的手順

Step1：データカウンターのAT回数を確認。直前にAT（ST）が1回以上あれば天井短縮の可能性あり。

Step2：前回の獲得枚数を確認。100枚未満ならST駆け抜けの可能性が高く、596G短縮濃厚。

Step3：サブ液晶を確認。桜背景なら4周期以内エピボ濃厚。海門城なら天井短縮確定かつ海門回想確定。

Step4：前回のAT回数が2回以上あれば景之ST到達の可能性。景之ST終了後も596G短縮。

Step5：スランプグラフを確認。差枚マイナスが大きいほど景之STループ率がUPしている可能性あり。

⑤ 黒煙ポイントシステム
━━━━━━━━━━

ST駆け抜け時に「黒煙ポイント」が加算されます。
このポイントは内部的に蓄積され、規定値に到達すると次回初当たり時に上位ST（海門回想・真景之ST・裏景之ST）に突入します。

▶ 蓄積量の推測方法

・ST駆け抜け回数：データカウンター履歴から。駆け抜け1回で+1〜3pt（推定）
・サブ液晶の黒煙エフェクト：エフェクト強度で蓄積度を示唆
・当日の稼働G数：1日1回程度の解放頻度（実戦値ベース）

▶ 解放時の期待値

黒煙ポイント解放時は上位STへの突入が確定するため、通常の初当たりと比べて期待獲得枚数が大幅にUPします。

・海門回想突入：約50%。平均獲得枚数約1,500枚
・真景之ST突入：約30%。平均獲得枚数約2,000枚
・裏景之ST突入：約20%。平均獲得枚数約2,500枚

（3000万G集計からの推定値。公式解析とは異なる可能性あり）

▶ 活用法

データカウンターで「AT回数が多いのに出玉が少ない台」を探す。
→ AT1回あたりの平均獲得枚数が150枚以下 = ST駆け抜けが多い = 黒煙ポイント蓄積台。

例：AT回数4回・総出玉500枚（1回平均125枚）の台は、黒煙ポイント蓄積が濃厚。
通常の期待値に+3,000〜5,000円を上乗せして判断できます。

⑥ 景之STループ率と差枚数の相関
━━━━━━━━━━

3000万Gのデータ分析から見えてきた最大の発見です。

景之STのループ率は公式には「設定差あり」としか発表されていません。
しかし大量の実戦データを集計すると、差枚数とループ率に相関がある可能性が浮上しています。

・差枚+1000枚以上：ループ率 約28%（基準値）
・差枚±500枚以内：ループ率 約32%（+4pt）
・差枚-500〜-1000枚：ループ率 約38%（+10pt）
・差枚-1000枚以下：ループ率 約45%（+17pt）

（設定の偏りを含む可能性あり。参考値として活用してください）

▶ 活用法

差枚マイナスが大きい台で景之STに突入した場合、ループ率が通常より10pt以上高くなる。
つまり「差枚凹み台の景之ST後は通常以上に粘る価値がある」ということです。

この判断の差が、月に2〜3万円の収支差を生みます。

⑩ やめどき完全ガイド
━━━━━━━━━━

▶ ST非当選（エピソードボーナスのみ）の場合
→ ST終了後20G間は「カバネリ高確」状態。サブ液晶と輪廻くじを確認してから判断。
→ サブ液晶が桜or海門城に変化 → 絶対にやめない（次回4周期以内確定）
→ 輪廻くじが大吉or中吉 → 次周期まで続行
→ それ以外 → 20G打って確認後にやめ

▶ 通常ST終了後
→ 次回は天井596Gに短縮。サブ液晶を確認。
→ 桜or海門城 → 続行
→ 通常背景 → やめどき判断（他に高期待値台があるか次第）

▶ 景之ST終了後
→ 596G短縮＋景之STループ抽選あり。
→ 差枚マイナスが大きい → ループ率UP。次回ST当選まで続行推奨。
→ 差枚プラスの場合 → 通常のやめどき基準に従う。

▶ AT後の閉店考慮やめどき
→ 残り3時間以上：通常通りの判断
→ 残り2時間：短縮台なら続行、通常天井は550G以上のみ
→ 残り1時間：短縮台の450G以上のみ。それ以外はやめ

▶ ケース1：596G短縮台を30秒で判別して+16,000円

【状況】14時頃。カバネリ海門決戦が500Gハマり。データカウンター：AT回数1回、前回獲得80枚。サブ液晶：通常背景。

【判断】AT回数1回あり。前回獲得80枚＝ST駆け抜け濃厚。→ 天井596G短縮の可能性大。500Gなら残り約96G。→ 打つ。期待値+9,500円以上。

【結果】520Gで4周期目到達 → エピソードボーナス → 景之ST突入 → 1,200枚獲得。
一般人なら「500Gだから拾おう」で同じ行動をしたかもしれません。でも一般人は「なぜこの台を拾うべきか」の根拠を持っていない。通常996G天井の500Gだったら期待値は+1,800円しかありません。


▶ ケース2：サブ液晶「海門城」を発見して+39,600円

【状況】10時頃、朝一巡回中。カバネリ海門決戦が120Gハマり。サブ液晶：海門城。

【判断】海門城＝天井596G短縮確定＋海門回想確定。120Gという「普通ならスルーするG数」でも、海門城が見えた瞬間に最優先。期待値+10,000円以上。

【結果】380Gで4周期目到達 → エピソードボーナス → 海門回想 → 景之STループ3回 → 2,500枚獲得。


▶ ケース3：黒煙蓄積台を推測して+38,000円

【状況】17時頃。350Gハマり。データカウンター：AT回数4回、総出玉500枚（1回平均125枚）。スランプグラフ：右肩下がり。

【結果】400Gで周期到達 → 黒煙ポイント解放 → 真景之ST突入 → 2,000枚獲得。
「350Gハマり」だけ見たら打たない。でも「AT4回で500枚」が読めれば、判断が180度変わります。
